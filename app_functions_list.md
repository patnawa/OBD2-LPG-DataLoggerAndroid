# OBD2 Petrol/LPG/CNG Data Logger Android — Features & Functions List
**รายการฟังก์ชันการทำงานและคุณสมบัติทั้งหมดของโปรแกรม**

---

## 1. OBD2 Data Logging & Recording (การบันทึกข้อมูล OBD2)
- **40+ OBD2 PIDs Support**: อ่านและวิเคราะห์ข้อมูลเซนเซอร์มาตรฐาน OBD2 กว่า 40 ค่า เช่น RPM, Speed, Fuel Trims, Coolant Temp, Intake Temp, MAF/MAP, Engine Load, Fuel Level, Ethanol %, O2 Sensors, Lambda (Wideband) และแรงดันไฟกล่องควบคุม
- **Dual-Format Logging**: บันทึกข้อมูลพร้อมกันในรูปแบบไฟล์ CSV (รองรับมาตรฐานสากล RFC-4180) และไฟล์ JSONL (รูปแบบวัตถุ JSON หนึ่งชุดต่อหนึ่งบรรทัด เหมาะสำหรับงานโปรแกรมมิ่งและ AI)
- **Custom Log Folder (Storage Access Framework - SAF)**: เลือกพื้นที่บันทึกไฟล์ข้อมูลได้อิสระ ไม่ว่าจะเป็นในเครื่อง, การ์ดหน่วยความจำภายนอก (SD Card), แฟลชไดร์ฟ (USB OTG) หรือ Google Drive
- **Fuel-Mode Prefixed Filenames**: ตั้งชื่อไฟล์ระบุโหมดเชื้อเพลิงอัตโนมัติเพื่อป้องกันความสับสน เช่น `PETROL_20260705_120000_obd2.csv` และ `LPG_20260705_130000_obd2.csv`
- **Background Foreground Service**: บันทึกข้อมูลได้ต่อเนื่องแม้ปิดหน้าจอหรือสลับไปใช้แอปพลิเคชันอื่น ด้วยบริการเบื้องหน้า (Foreground Service พร้อมแจ้งเตือนแบบยึดแน่น) ป้องกันไม่ให้ Android OS ปิดแอป
- **Dynamic PID Autodetection**: ค้นหารหัส PIDs ที่รถยนต์เครื่องนั้นๆ รองรับจริงผ่านการถอดรหัสบิตแมป OBD2 PIDs (0x00, 0x20, 0x40) ป้องกันการส่งคำสั่งที่ไม่รองรับ ช่วยลดการหน่วงของสัญญาณลงได้ 30-50%

## 2. LPG/CNG Tune Assist & Fuel Trim Mapping (ระบบแผนที่การวิเคราะห์และปรับจูนแก๊ส)
- **Live Fuel Map (2D Grid Matrix)**: ตารางวิเคราะห์การจ่ายเชื้อเพลิงสดแบบ 2 มิติ (เวลาฉีดน้ำมัน T.inj ms × รอบเครื่องยนต์ RPM) พร้อมแสดงผลรวมค่า Fuel Trim (STFT + LTFT) แยกสีชัดเจนจาก -25% (สีแดง-หนาเกิน) ถึง +25% (สีน้ำเงิน-บางเกิน) และสีเขียว (สมบูรณ์แบบ)
- **Dual-Fuel Baseline Comparison**: แยกจัดเก็บเลเยอร์ข้อมูลสำหรับการขับขี่ด้วยน้ำมัน (Petrol) และแก๊ส (LPG/CNG) ไว้แยกกัน เพื่อนำมาเทียบหาค่าส่วนเบี่ยงเบน
- **Deviation Matrix**: คำนวณร้อยละความเบี่ยงเบนของค่า Trim ระหว่างแก๊สและน้ำมันในแต่ละช่องอัตโนมัติตามสูตร:
  $$\text{Deviation} = \text{LPG Trim} - \text{Petrol Trim}$$
- **Tune Assist Correction Grid**: ประมวลผลและแสดงตัวเลขอัตราส่วนร้อยละที่แนะนำให้ผู้ปรับจูนป้อนเข้าไปบวก (+) หรือลบ (-) ในตารางตัวคูณ (Multiplier Map) ของกล่องจูนแก๊สเพื่อให้แก๊สจ่ายได้เท่ากับน้ำมัน
- **Gated Plotting**: เก็บสถิติ Trim เฉพาะในช่วงระบบจ่ายน้ำมันแบบควบคุมป้อนกลับ (Closed Loop เท่านั้น) และมีอุณหภูมิน้ำหล่อเย็นเครื่องยนต์ (ECT) $\ge 80^\circ\text{C}$ เพื่อความเที่ยงตรงสูง
- **Gold Border Cell Locking**: ระบบกรองค่าชั่วคราว (Transient Filter) โดยจะทำการล็อกค่าในตาราง (กรอบสีทอง) ก็ต่อเมื่อรถวิ่งแช่ ณ รอบและภาระโหลดนั้นๆ ครบ 20ครั้งขึ้นไป
- **Log Import/Export**: ส่งออกแผนที่ตัวคูณชดเชยเป็นไฟล์ CSV เพื่อนำไปใช้จูนแก๊สผ่านคอมพิวเตอร์ และรองรับการนำเข้าไฟล์ CSV ย้อนหลังเพื่อเปิดดูแผนที่ออฟไลน์หรือเปรียบเทียบไฟล์ข้อมูลน้ำมันและแก๊สต่างช่วงเวลากันได้

## 3. Live Telemetry & Widgets Dashboard (หน้าปัดแสดงผลแบบเรียลไทม์)
- **Custom Analog Gauges (4 Slots)**: เกจวัดทรงกลมแบบเข็มกวาดเสมือนจริง มีแถบสีแบ่งโซนอันตราย ตั้งค่าเตือน (Warning Limit) บันทึกค่าสูงสุด (Peak Value) และเลือกธีมสีแยกตามประเภทข้อมูล (RPM=แดง, Speed=ฟ้า, Temp=เหลือง, Load=เขียว)
- **Digital Text Cards (4 Slots)**: การ์ดตัวเลขอ่านค่าเซนเซอร์ขนาดใหญ่ แตะเพียงครั้งเดียว (Single-tap) เพื่อเปลี่ยนพารามิเตอร์หรือปิดสล็อตได้สะดวกผ่านรายการเลือกแบบ Bottom Sheet
- **Real-Time Rolling Graphs (5 Slots)**: กราฟเส้นแสดงทิศทางข้อมูลเซนเซอร์ย้อนหลังแบบเรียลไทม์ ปรับสเกลแกน Y อัตโนมัติตามค่าต่ำสุด-สูงสุด
- **Full Sensor Table**: ตารางรายงานค่าสถานะ OBD2 ทุกตัวที่รถรองรับพร้อมหน่วยวัดและสถานะตอบรับ
- **Smart OBD2 Tuning Analyzer**: ระบบวิเคราะห์ค่า Trim อัตโนมัติ บอกอุณหภูมิน้ำหล่อเย็น (ECT), สถานะจ่ายเชื้อเพลิง (Loop Status: Open/Closed), สรุปค่า STFT/LTFT ปัจจุบัน และคำแนะนำสำหรับคนขับแบบสั้น (เช่น แนะนำให้ขับเก็บข้อมูลน้ำมันเพิ่ม หรือเตือนแก๊สจ่ายบางไป)

## 4. OBD2 Diagnostics & Troubleshooting (การตรวจวินิจฉัยและสแกนวิเคราะห์โค้ด)
- **DTC Scanner (Mode 03, 07, 0A)**: สแกนอ่านรหัสข้อผิดพลาดของรถยนต์ทั้งรหัสที่บันทึกแล้ว (Stored), รหัสที่กำลังรอสะสมค่า (Pending) และรหัสถาวร (Permanent) พร้อมแสดงรายละเอียดคำอธิบายข้อผิดพลาด
- **DTC Clear (Mode 04)**: ล้างรหัสข้อผิดพลาดและดับไฟเตือนรูปเครื่องยนต์บนหน้าปัด (MIL)
- **Mode 06 Monitor Test Results**: ดึงค่าประเมินผลการตรวจสอบตัวเองของชิ้นส่วนที่ไม่ได้ทำงานต่อเนื่อง เช่น ประสิทธิภาพแคทตาไลติก, อ๊อกซิเจนเซนเซอร์, ระบบหมุนเวียนไอเสีย (EGR), ระบบไอน้ำมันเชื้อเพลิง (EVAP) และอัตราการจุดระเบิดบกพร่อง (Misfire) พร้อมขีดจำกัดขั้นต่ำ/สูงสุด
- **Freeze Frames (Mode 02)**: เรียกดูภาพนิ่งข้อมูลรถยนต์ขณะเกิดรหัสข้อผิดพลาดขึ้น (ความเร็ว, รอบเครื่อง, ความร้อน, ฯลฯ) เพื่อวิเคราะห์ต้นเหตุของปัญหา
- **DTC Enrichment database**: ฐานข้อมูลรหัสปัญหากว่า 150 รายการ แสดงสาเหตุที่เป็นไปได้, แนวทางการซ่อมแซม และรอบการทดสอบของกล่อง ECU เพื่อประหยัดเวลาช่าง
- **Emissions Readiness Monitors**: รายงานสถานะความพร้อมของระบบควบคุมมลพิษ 12 ระบบ เพื่อเตรียมนำรถเข้าตรวจสภาพไอเสียประจำปี
- **VIN & Calibration Info (Mode 09)**: อ่านรหัสตัวถังรถ (VIN) 17 หลัก และรหัสเวอร์ชันซอฟต์แวร์กล่อง ECU (Cal-ID / CVN)

## 5. Power Monitor & Battery Charging Tester (ระบบตรวจวิเคราะห์แบตเตอรี่และระบบไดชาร์จ)
- **Smart Voltage Acquisition**: อ่านแรงดันไฟเลี้ยงผ่านรหัสมาตรฐาน OBD2 PID `01 42` (Control Module Voltage) และหากรถรุ่นเก่าไม่รองรับ ระบบจะสลับไปส่งคำสั่งฮาร์ดแวร์ตรง **`AT RV`** เพื่อวัดค่าแรงดันอนาล็อกจากพิน 16 ของช่อง OBD2 ทันที การันตีการอ่านค่าได้สำเร็จกับรถยนต์ทุกคัน 100%
- **11 Automated Diagnostic Tests**: โปรแกรมตรวจประเมินอัตโนมัติ 11 รายการหลัก:
  - **State of Charge (SoC%)**: ประเมินปริมาณไฟฟ้าที่เก็บอยู่ในแบตเตอรี่เทียบตามประเภทเคมี
  - **State of Health (SOH)**: คำนวณความเสื่อมสภาพของแบตเตอรี่จากอัตราความต้านทานภายในและการตอบสนองต่อโหลด
  - **Battery Life Estimate**: คาดการณ์จำนวนเดือนคงเหลือของแบตเตอรี่ โดยคำนวณจากประเภทแบตเตอรี่, อายุการใช้งาน และปัจจัยสภาพภูมิอากาศเขตร้อน (Tropical Climate Factor)
  - **Alternator Output**: ตรวจสอบแรงดันไดชาร์จขณะเครื่องยนต์ทำงานรอบเดินเบา
  - **Voltage Drop Test**: วัดการสูญเสียแรงดันไฟขณะเปิดเครื่องใช้ไฟฟ้าในรถพร้อมกัน (แอร์, พัดลมหม้อน้ำ, ไฟหน้า, ไล่ฝ้า)
  - **Cranking Voltage**: บันทึกแรงดันตกต่ำสุดขณะสตาร์ทเครื่องยนต์ (ใช้การสุ่มตรวจความเร็วสูง 80ms)
  - **Alternator Ripple/Diode Test**: ตรวจสอบกระแสไฟฟ้าสลับ (AC ripple) เล็ดลอดเข้าระบบ เพื่อหาไดโอดชำรุดในตัวปั่นไฟ
- **Multi-Chemistry Support**: รองรับวิเคราะห์เคมีแบตเตอรี่ 6 ประเภท: Flooded (กรด-ตะกั่วทั่วไป), AGM (ใยแก้วกักกรด), EFB, Gel, Calcium, และ LiFePO4 (ลิเธียมฟอสเฟต)
- **Real-Time Voltage Analyzer Badge**: แท็กสีวิเคราะห์สถานะแรงดันไฟอัตโนมัติ (กำลังชาร์จ / ชาร์จเกินขนาด / ดับเครื่องปกติ / แรงดันไฟสแตนด์บาย / แรงดันต่ำโหลดหนัก / วิกฤตคายประจุลึก)

## 6. External AI Agent Integration & HTTP API Server (ระบบเชื่อมต่อ AI และ API ภายนอก)
- **Embedded REST API Server (NanoHTTPD - Port 8080)**: เปิดเซิร์ฟเวอร์ย่อยในโทรศัพท์ผ่านวง WiFi เดียวกัน เพื่อส่งสัญญาณข้อมูลเซนเซอร์ OBD2 ทั้งหมดเป็นไฟล์ JSON ภายนอก รองรับการเชื่อมต่อกับ AI Agents, ระบบคอมพิวเตอร์ หรือ MCP Clients
- **API Endpoints**:
  - `GET /api/ping` — ตรวจสถานะเซิร์ฟเวอร์และค่าเวลาระบบ
  - `GET /api/status` — ส่งสถานะการเชื่อมต่อรถยนต์, รหัส VIN และโหมดเชื้อเพลิงปัจจุบัน
  - `GET /api/data` — รายงานพารามิเตอร์เซนเซอร์ OBD2 ทั้งหมดที่เป็นปัจจุบันพร้อมหน่วยวัด
  - `GET /api/map` — ส่งออกข้อมูลตารางจูนแก๊ส 2 มิติย้อนหลังทั้งหมด
  - `GET /api/map/summary` — ส่งรายงานสรุปพิกัดเซลล์ที่แก๊สเพี้ยนสูงสุด และคำแนะนำจูนแก๊สชดเชย
  - `GET /api/map/export` — ดาวน์โหลดไฟล์แผนที่ตัวคูณแก๊สในรูปแบบ CSV โดยตรง
- **CORS Enabled**: อนุญาตการดึงข้อมูลจากเว็บเบราว์เซอร์หรือเครื่องแล็ปท็อปภายนอกโดยไม่ติดขัดปัญหาความปลอดภัย
